## INSTALLATION

1. Ensure PIL/Pillow is installed:
   pip install pillow

2. Install cryptography:
   pip install cryptography

3. Run:
   python main.py

## NEW DEPENDENCIES

- Pillow (PIL fork) - for logo image handling
